All instructions are three length in size

Valid instructions 
	lda , add, orr, sta, not, and, hlt, out, sub

lda, add, and, orr, sta, sub require two more parameters (address in hexadecimal Eg : 0x5 and value to be written at that address)
							 (Example : lda 0x5 #35 ; loads 35 into accumulator)
